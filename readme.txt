# Calculadora em Python

Este projeto é uma simples calculadora desenvolvida em Python que realiza as quatro operações matemáticas básicas: adição, subtração, multiplicação e divisão. A calculadora permite ao usuário escolher a operação que deseja realizar e depois inserir dois números para obter o resultado.

## Funcionalidades

- Soma: Adiciona dois números.
- **Subtração:** Subtrai o segundo número do primeiro.
- **Multiplicação:** Multiplica dois números.
- **Divisão:** Divide o primeiro número pelo segundo. Se o segundo número for zero, a calculadora retorna uma mensagem de erro indicando que não é possível dividir por zero.

## Como usar

1. Ao executar o script, você será solicitado a escolher uma operação dentre as quatro opções:
   - 1. Soma
   - 2. Multiplicação
   - 3. Subtração
   - 4. Divisão

2. Após escolher a operação, você deverá inserir dois números quando solicitado.

3. O resultado da operação será exibido na tela, indicando o cálculo realizado.

4. A calculadora executa uma operação por vez e encerra após a primeira execução.

```python

3. Subtração
4. Divisão

Digite sua escolha (Opções 1/2/3/4): 1
